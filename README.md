# The Big Hotel
HM of HuckerU

## About

- Ts project
- Express server with mongoose model,
- MongoDB - CRUD operations,
- controllers, 
- routes,
- Ui

There are three collections:
1. Guests
2. Reservations
3. Rooms

**CRUD** operations:
 - [x] Insert
 - [x] Find by id 
 - [x] Find all 
 - [x] Update and save. 
 - [x] Delete

If you get to any route that does not exist, you will get a 404 page.
